### 光源类型

### 反射

漫反射、环境反射

#### 漫反射

漫反射光颜色 = 入射光颜色 * 表面基底色 * cosx

漫反射个方向是均匀的，强度相等

#### 环境反射

针对环境光而言，各方向均匀、强度相等

环境反射光颜色 = 入射光颜色 * 表面基底色


表面的反射光颜色 = 漫反射光颜色 + 环境反射光颜色
